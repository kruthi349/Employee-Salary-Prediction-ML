# 💰 Employee Salary Prediction using Machine Learning

## 📌 Project Overview

This project predicts an employee's annual salary based on professional and workplace-related attributes using Machine Learning.

A modern web application was developed using **Flask**, **HTML**, **CSS**, and **JavaScript** to allow users to enter employee information and receive an instant salary prediction.

---

## 🚀 Features

- Employee Salary Prediction
- Interactive Web Interface
- Machine Learning Model Integration
- Flask Backend API
- Responsive UI
- Professional Dashboard
- Real-time Prediction

---

## 📂 Dataset

The dataset contains employee information including:

- Job Title
- Experience Years
- Education Level
- Skills Count
- Industry
- Company Size
- Location
- Remote Work
- Certifications
- Salary (Target Variable)

## ▶️ How to Run

### Clone Repository

```bash

```

### Open Project

```bash
cd Employee-Salary-Prediction-ML
```

### Install Requirements

```bash

```

### Run Flask

```bash
python app.py
```

